# easyp 0.1.3-alpha1 Release

We're excited to announce the alpha release of easyp 0.1.3, featuring significant improvements to our ring wrapper implementation and enhanced platform compatibility.

## What's New

### Ring Wrapper Enhancements
- **Platform-aware ring implementation**: Automatically selects the appropriate ring crate based on target platform
- **Redox compatibility**: Seamless support for Redox OS using the redox-specific ring fork
- **Unified API**: Consistent ring API across all supported platforms
- **Improved build system**: Better dependency management for ring-related crates

### Technical Improvements
- **Enhanced crypto support**: Full ring feature set available including alloc, signature, and digest algorithms
- **Better error handling**: Improved error propagation in crypto operations
- **Performance optimizations**: Reduced overhead in crypto wrapper operations

### Updated Dependencies
- Updated ring wrapper to use latest upstream ring (0.17.14) with full feature support
- Enhanced redox ring fork integration with latest commit (fa98b490)
- Improved compatibility with rustls and webpki ecosystems

## Installation

For the latest development version:

```bash
curl www.easyp.net/easyp-0.1.3-alpha1-x64.gz -o easyp.gz && gunzip easyp.gz && [  "$(sha256sum < easyp)" = "UPDATE_CHECKSUM_HERE  -" ] && chmod +x easyp && sudo ./easyp
```
