Simple

Easy to use with minimal configuration required.

Just run easyp and you're ready to serve HTTPS with automatic certificate management.

Tiny

Minimal memory footprint and small binary size.

Uses only 8.1 MB RSS compared to Apache's 36 MB. The entire binary is just 2.8MB.

Rusty

Built with Rust for memory safety and performance.

Aiming for 100% pure Rust, with acme-lib rewritten to use rusty rustls.
